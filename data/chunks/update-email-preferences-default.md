## Context

This process explains how to set a default selection for email preferences on the HWL platform, where users are automatically subscribed to all available email notifications by selecting the default option.

## Response

To set email preferences to default on HWL and subscribe to all available notifications, follow these steps:
1. From the HWL menu on the left under the Settings tab, select 'Email Preference'.
2. From the right, select 'Default to Select All Emails'.
3. In the pop-up message that appears, select 'OK'.
4. The preferences will automatically be set. Emails that will be sent are indicated as gray within the interface, while a flashing red message will display the text 'All Emails Selected by Default' to confirm the default settings have been applied.
5. After this, you will need to set Job Titles Subscribed.
6. Click 'OK' on the subsequent pop-up message.
7. Select 'Default to Select All Job Titles'.
